My Interest

Front-end web developers, the "artists" formerly known as web designers are the bunch of people in the company that make sure that the data coming from the backend gets displayed on the browser. They also make sure it looks as closely as possible as the design, that , CED came up with, and that the user can navigate through it, accessing the data. They don't need to know about complex OO programming, tricky database structures or how a path is done in Illustrator. They get data from the backend people and an example how the result looks like from CED. All they need to do is to hack some HTML or XSLT together and link the different documents. Piece of cake.
